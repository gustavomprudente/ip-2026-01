📘 Lista 01 - Exercícios em Go

Esta pasta contém as resoluções da **Lista 01** de exercícios utilizando a linguagem Go.

📂 Estrutura

Cada arquivo representa uma questão:

* "q01.go" → Questão 01
* "q02.go" → Questão 02
* "q03.go" → Questão 03
* ...

🎯 Objetivo

Praticar conceitos básicos de programação em Go:

* Entrada e saída de dados
* Variáveis
* Estruturas de repetição
* Condições
* Cálculos (como média, soma, etc.)

## 🧠 Observação

Os exercícios são voltados para aprendizado, então o foco é clareza e prática dos conceitos.
